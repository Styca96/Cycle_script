# Cycle_script

Script for execute sequential command on multiple Instrument:

- Climate Chamber (modbus protocol)
- ITECH dc bidiretional load (SCPI protocol)
- CHROMA grid simulator (SCPI protocol)
- Arm-Xl (SSH protocol)
